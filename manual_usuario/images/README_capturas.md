# Guía de capturas — Manual de Usuario eTax2

Las imágenes de la carpeta `images/` son **recreaciones fieles** de la interfaz. Si quieres usar **capturas reales**, toma cada pantalla en la URL indicada y **reemplaza el archivo conservando el mismo nombre** (formato PNG recomendado, ancho ≈ 1460 px).

| Archivo | Pantalla a capturar | URL |
|---|---|---|
| `01-dashboard.png` | Dashboard / Estado Financiero | `/dashboard` |
| `02-cxc-facturas.png` | Cuentas por Cobrar → Facturas (listado) | `/admin/cxc/facturas` |
| `03-nueva-factura-venta.png` | Ventas → Facturas → Nueva factura (formulario) | `/admin/ventas/facturas` → *Nueva factura* |
| `04-liquidacion-itbms.png` | Reportes → Liquidación ITBMS | `/admin/reportes/liquidacion-itbms` |
| `05-centro-ayuda.png` | Centro de Ayuda | `/admin/ayuda` |
| `06-navegacion.png` | Dashboard con elementos de navegación señalados | `/dashboard` |

## Sugerencias para capturas reales

- Usa la compañía con datos representativos (por ejemplo *ETAX 2*).
- Captura a pantalla completa del navegador, con el menú lateral visible.
- Si compartirás el manual fuera de la empresa, evita mostrar cifras o nombres de clientes sensibles (usa una compañía de demostración).
- Mantén la misma proporción para que las imágenes se vean uniformes en el documento.
